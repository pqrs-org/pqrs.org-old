How to build

Requirements

  • devkitPro r18 (buildscripts 20060412) (You need to use version r18. Do not use the latest release!)

Steps

Execute the following commands in Terminal. 

% tar ztf vulkanon-2.1.tar.gz
% cd vulkanon-2.1/vulkanon/
% make
