Controls

At Main Menu

A Button
Choose.
B Button
Cancel.
Control Pad
Move cursor.

In Game

B Button
Back to Main Menu.
R Button
Skip to next barrage. (You cannot skip in Shooting Mode.)
Start,Select
Toggle slow mode.
Control Pad
Move.

Modes

Select Barrage
Select the barrage from the list.
Shooting Level 1-4
Destroy the boss.
BulletsMorph
Generate the barrage automatically.

Slow Mode

You can toggle slow mode by START or SELECT.

  • Bullets will be slowed down to 1/4 speed.
  • The color of the bullets becomes purple.

You need "Jewel" to enter slow mode. Slow mode will be canceled when jewel becomes zero or you are bombarded.
