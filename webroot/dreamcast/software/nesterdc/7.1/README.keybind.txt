keybinds 

========================================
in main menu
 * A: ROM menu
 * X: option menu
 * B: VMU menu
 * Y: credit

 * START + Y + L + R: reset. 


========================================
in ROM menu
 * A: move to the directory 
 * Y: move to the parent directory
 * X + UP/DOWN/LEFT/RIGHT: high speed cursor move
 * analog: adjust cursor wait


========================================
in playing game

[Dreamcast Controller]
 * X: NES B
 * A: NES A
 * Y: NES SELECT
 * START: NES START
 * UP/DOWN/LEFT/RIGHT: same as DC
 * L: toggle-autofire-mode
   (you can check current autofire status at VMU)

 * L + R + START: return to main menu

 * analog: change frameskip
   - UP:        'no skip'
   - DOWN:      'auto x2.0'
   - RIGHT:     'auto x1.0'
   - LEFT:      'auto x1.5'
   - LEFT + R:  'auto x4.0'
   (you can check current frameskip rate at VMU)

 * B + R: quick save menu
   items
    - quick state save (clear when exit game)
    - quick state load 

 * Y + R: VMU state save menu
   items
    - VMU state save 
    - VMU state load

 * X + R: system menu 
   items
    - reset
    - fds set diskside

 * A + R: option menu
   items
    - toggle autofire enable/disable when autofire-mode
    - adjust screen position
    - toggle bilinear filter on/off

 * START + R: cheat menu 
    - load Game Genie code
    - load NNNesterJ cheat code
    - set NNNesterJ cheat code checkflag


[Arcade Stick]
 * X: NES B
 * Y: NES A
 * C: toggle-autofire-mode
   (you can check current autofire status at VMU)
 * Z: NES SELECT
 * START: NES START
 * A: NES START
 * UP/DOWN/LEFT/RIGHT: same as DC
