* How to use Game Genie

 1. You must burn CD with Game Genie code. like

     /1ST_READ.BIN
     /pics/credits.bmp
     /pics/...
     /games/super_mario_3.nes
     /games/...
     /genie/super_mario_3/high_jump
     /genie/super_mario_3/high_speed
     /genie/...

    (You don't have to sync "ROM name" and "genie name"). 

 2. press X + R in game, and select genie file. 


* Game Genie code format

  A plain text with a Game Genie code per line. like below

========================================
GZUXNGEI
SXEZSKOZ

========================================
